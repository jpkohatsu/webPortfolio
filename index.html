Jobiato app: Indeed and Meetup API's & Firebase

front end

Jobiato is a front end application that allows the user to search Indeed for job postings and will simultaneously find meetups in the area associated with the search terms. The user can then "like" jobs that interest them and have the jobs saved to their user account with the help of Google Firebase. The technologies used in this app were HTML, CSS, and Firebase for storage. Ajax calls were used to both the Indeed and Meetups Api's.
